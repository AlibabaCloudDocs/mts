# Request structure {#reference_y14_jvm_x2b .reference}

## Service address {#section_xxv_kvm_x2b .section}

For more information about the access addresses of the Media Processing APIs, see [Service region](https://help.aliyun.com/document_detail/43248.html).

## Communication protocols {#section_yxv_kvm_x2b .section}

Request communication over HTTP or HTTPS is supported. For higher security, we recommend that you send requests through HTTPS channels.

## Request method {#section_zxv_kvm_x2b .section}

The system supports HTTP GET requests. In this mode, the request parameters must be included in the request URL.

## Request parameters {#section_ayv_kvm_x2b .section}

For each request, the operation to be executed must be specified, namely the Action parameter. Each operation must contain the public request parameters and the specific request parameters of the specified operation.

## Character encoding {#section_byv_kvm_x2b .section}

Requests and responses are encoded using the UTF-8 character set.
