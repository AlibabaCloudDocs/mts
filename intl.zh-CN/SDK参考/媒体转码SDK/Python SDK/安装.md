# 安装 {#concept_msd_bbc_z2b .concept}

本文介绍阿里云Python SDK推荐的pip安装方式。

-   Python 2.x

    ```
    ```
    pip install aliyun-python-sdk-core
    pip install aliyun-python-sdk-mts
    ```
    ```

-   Python 3.x

    ```
    ```
    pip install aliyun-python-sdk-core-v3
    pip install aliyun-python-sdk-mts
    ```
    ```
